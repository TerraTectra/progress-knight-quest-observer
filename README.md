# Progress Knight Quest: Observer

An experimental continuation of Progress Knight Quest focused on a new Multiverse route and the Observer endgame layer.

Live build:
https://terratectra.github.io/progress-knight-quest-observer/

Current release:
v0.1.0-beta

## Added in this fork

- x5 baseline game speed with admin speed controls up to x1000.
- Multiverse unlock after the Void route.
- Passive real-time Multiverse Points that are not affected by Time Warp.
- Ten universes with progressively distorted rules, local parameters, passive MP weights, and universe breaks.
- Multiverse upgrades for XP, income, lifespan, Evil, Essence, Dark Matter, MP scaling, and late-route support.
- Observer as the third global layer after Universe X.
- Observer subjects that simulate their own Progress Knight runs from the start and generate Observer Points.
- Save migration, tab regression checks, long idle stress tests, and a unified release check runner.

This is a beta release. Balance feedback and bug reports are expected.
